Machine Learning-Based Object Situation Awareness Using a Monocular Camera For Autonomous Navigation of Multirotor Aerial Robots in Populated Areas
The goal of this project, undertaken as part of my Master's studies, was to develop a machine learning-based system for situation awareness in autonomous multirotor aerial robots using monocular cameras. The project aimed to detect and localize objects, estimate depth, and enable safe navigation in populated areas. The focus was on utilizing minimal equipment to create a cost-effective solution suitable for applications like logistics and emergency response..
Tools used: Python, OpenCV, TensorFlow, Keras, NumPy, Matplotlib, PyTorch
Techniques: Object Detection (YOLO, Mask R-CNN), Image Segmentation, Depth Estimation (Monodepth, Monodepth2), 3D Mapping (Octree, Octomap)
Platforms: Jupyter Notebook, Meshlab, ROS, Gazebo
Year: Apr 2020
